FEA Solver API

REST API for 3D static structural finite element analysis. Submit analysis jobs, retrieve stress and displacement results, and integrate FEA capabilities into your applications.

https://fea-solver.vercel.app

Introduction

The FEA Solver API provides programmatic access to a 3D static structural finite element analysis engine built on deal.II. Submit meshes, boundary conditions, and loads to receive detailed stress, displacement, and safety factor results.

The API supports isotropic and orthotropic materials, various boundary condition types, and multiple load types including pressure, point forces, gravity, and thermal loads.

Quick Start

Get started with your first FEA analysis in four steps.

1
Submit an Analysis Job
POST your mesh, boundary conditions, and loads to /api/analyze
2
Receive Job ID
The API returns a unique job ID to track your analysis
3
Poll for Status
Check progress at /api/jobs/{id} until status is "completed"
4
Retrieve Results
Get results at /api/jobs/{id}/results or download VTK files

Example Request

curl 
curl -X POST https://fea-solver.vercel.app/api/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "mesh": {
      "type": "box",
      "min": [0, 0, 0],
      "max": [100, 10, 10],
      "subdivisions": [20, 4, 4]
    },
    "materials": {
      "default": "steel_structural"
    },
    "boundary_conditions": [
      {
        "type": "fixed",
        "target": {"type": "boundary_id", "id": 0}
      }
    ],
    "loads": [
      {
        "type": "surface_force",
        "target": {"type": "boundary_id", "id": 1},
        "force_per_area": [0, 0, -1000000]
      }
    ]
  }'

Example Response

json
{
  "job_id": "fea_20260124_143052_847291",
  "status": "queued",
  "queue_position": 1,
  "message": "Analysis job submitted successfully"
}

Authentication

The API currently supports public access for evaluation purposes.

Rate Limiting
Public API is limited to 10 requests/minute and 100 requests/day per IP address.

Error Handling

The API uses standard HTTP status codes and returns error details in JSON format.

Code Description
200Success
202Accepted - Job submitted or still processing
400Bad Request - Invalid input
404Not Found - Job or resource not found
429Too Many Requests - Rate limit exceeded
503Service Unavailable

Error Response Format

json
{
  "error": "Validation failed",
  "details": ["Missing required field: mesh"]
}

Submit Analysis

POST /api/analyze Submit a new analysis job

Submits a new structural analysis job. Returns immediately with a job ID for tracking.

Request Body

ParameterTypeDescription
meshrequired object Mesh definition (box, cylinder, or file)
boundary_conditionsrequired array Array of boundary conditions
materialsoptional object Material assignments (defaults to steel)
loadsoptional array Array of load definitions
solver_optionsoptional object Solver configuration
unitsoptional object Unit system (SI, SI_MM, US_CUSTOMARY)

Response

202 Job accepted
400 Validation error

Job Status

GET /api/jobs/{job_id} Get job status and progress

Returns the current status of an analysis job.

Job Status Values

StatusDescription
queuedWaiting in queue
runningCurrently executing
completedFinished successfully
failedEncountered an error
cancelledCancelled by user

Example Response

json
{
  "job_id": "fea_20260124_143052_847291",
  "status": "running",
  "progress": 0.45,
  "current_stage": "Assembling system matrix"
}

Job Results

GET /api/jobs/{job_id}/results Get analysis results

Returns complete analysis results when job status is "completed".

Example Response

json
{
  "job_id": "fea_20260124_143052_847291",
  "status": "completed",
  "displacements": {
    "max": {"x": 0.00012, "y": 0.00024, "z": -0.00856}
  },
  "stress": {
    "von_mises": {"max": 125000000, "avg": 45000000}
  },
  "reactions": {
    "total_force": [0, 0, 10000]
  },
  "safety_factors": {
    "min": 2.0,
    "avg": 5.5
  }
}

Download Files

GET /api/jobs/{job_id}/files/{filename} Download output files

Downloads output files from a completed analysis.

Available Files

FilenameDescription
results.vtuVTK file for ParaView visualization
results.csvNodal results in CSV format
results.jsonComplete results in JSON

Cancel Job

DELETE /api/jobs/{job_id} Cancel a queued job

Cancels a job that is still in the queue. Running jobs cannot be cancelled.

Mesh Quality

POST /api/mesh/quality Analyze mesh quality

Performs mesh quality analysis without running a full FEA analysis.

Example Response

json
{
  "num_elements": 3200,
  "num_nodes": 14641,
  "metrics": {
    "jacobian_ratio": {"min": 0.85, "avg": 0.95},
    "aspect_ratio": {"max": 2.5, "avg": 1.8}
  },
  "quality_acceptable": true
}
